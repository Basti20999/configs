# Minecraft Server Configurations

A comprehensive collection of optimized configuration files for Minecraft servers running Paper, Purpur, Leaf, and related server software.

## 📋 Contents

This repository contains configuration files for:

- **Paper** - `paper-global.yml`, `paper-world-defaults.yml`
- **Purpur** - `purpur.yml`
- **Leaf** - `leaf-global.yml`
- **Spigot** - `spigot.yml`
- **Bukkit** - `bukkit.yml`
- **Server Properties** - `server.properties`
- **Pterodactyl Integration** - `egg-leaf-m-c.json` (Pterodactyl egg configuration)

## 🎯 Purpose

These configuration files are optimized for running high-performance Minecraft servers with enhanced features, performance tweaks, and stability improvements across different server software flavors.

## 📦 Usage

1. Clone this repository or download the configuration files
2. Place the appropriate config files into your server directory
3. Adjust settings as needed for your specific server requirements
4. Restart your server to apply the configurations

## 🔧 Compatibility

Configurations are provided for:
- **Paper** - Popular performance-optimized Bukkit fork
- **Purpur** - Highly customizable Paper fork
- **Leaf** - High-performance Paper fork with Purpur features and integrated Gale enhancements
